Svedea Navigation – Arbetsprov

Denna sida demonstrerar en responsiv och tillgänglig navigation byggd med vanilla HTML, CSS och JavaScript. Menydata renderas dynamiskt från ett JS-objekt i nav-data.js.

Prova att öppna Försäkringar i desktopmenyn, eller klicka på hamburgerikonen på mobil.

Källkod och README på GitHub.
